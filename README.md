# Smart City Weather Dashboard 🌦️

## 📌 Description

This project is a weather dashboard web application that provides real-time weather data and smart suggestions based on weather conditions.

## 🚀 Features

* Search weather by city
* Real-time weather data using OpenWeatherMap API
* 5-day forecast
* Smart suggestions (what to wear/carry)
* Responsive design

## 🔧 Technologies Used

* HTML
* CSS
* JavaScript (Fetch API)
* OpenWeatherMap API

## 📡 API Used

https://openweathermap.org/api

## ▶️ How to Run

1. Clone the repository
2. Add your API key in script.js
3. Open index.html in browser
